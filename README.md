## Gin + MySQL

### Setup and install

```js
go mod init gin/mysql
```

```js
go get -u github.com/gin-gonic/gin
```

```js
 go get -u github.com/go-sql-driver/mysql
```

```js
go install github.com/cosmtrek/air@latest
```

```js
air init
```

### Run auto reload

```js
air run main.go
```

### Run

```js

go run main.go
```
